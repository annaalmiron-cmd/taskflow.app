## 🚀 Sobre o projeto

O TaskFlow é uma aplicação fictícia criada para auxiliar estudantes na organização de tarefas diárias.

O objetivo deste projeto é praticar:

- Git
- GitHub
- Estruturação de projetos
- Organização de repositórios
- Versionamento

---

## ✨ Funcionalidades

✅ Adicionar tarefas  
✅ Excluir tarefas  
✅ Marcar tarefas como concluídas  
✅ Interface simples e intuitiva  
✅ Layout responsivo
